## 💡 Consejos

### Navegación Rápida
- Usa `Home` y `End` para saltar al inicio/final del cómic instantáneamente
- Usa `Ctrl` + `G` para saltar a una página específica

### Pantalla Completa
- `F11`: Pantalla completa estándar (muestra barra de tareas de Windows)
- `F12`: Pantalla completa inmersiva (oculta todo, experiencia máxima)
- `Esc`: Sale de cualquier modo de pantalla completa

### Zoom Inteligente
- `F`: Ajusta toda la imagen a la pantalla (mantiene proporciones)
- `W`: Ajusta solo el ancho (útil para manga/cómics verticales)
- `H`: Ajusta solo la altura (útil para páginas dobles)

### Modos de Lectura
- **Modo Noche** (`N`): Invierte colores para lectura nocturna
- **Modo Sepia** (`Shift+S`): Tono cálido para lectura prolongada
- **Modo Lectura** (`M`): Oculta distracciones

### Atajos Estilo Vim
Si vienes de Vim/Vi, puedes usar:
- `J`: Avanzar página
- `K`: Retroceder página
- `H`: Ajustar altura (en vez de izquierda)
- `W`: Ajustar ancho (en vez de palabra)

### Atajos con Alt
Los atajos con `Alt` te dan acceso rápido a ventanas secundarias:
- `Alt` + `F`: Favoritos
- `Alt` + `L`: Biblioteca
- `Alt` + `S`: Estadísticas
- `Alt` + `H`: Home

## 🎮 Configuración de Atajos

Puedes personalizar algunos comportamientos en **⚙️ Configuración**:

- **Dirección de lectura**: Cambia si las flechas van hacia adelante o atrás
- **Barra espaciadora**: Activa/desactiva avance con espaciador
- **Scroll continuo**: Cambia comportamiento de flechas arriba/abajo
- **Salir con ESC**: Configura comportamiento de la tecla ESC en pantalla completa
