# 🚀 NUEVAS CARACTERÍSTICAS - Percy's Library

## Versión 2.0 - Actualización Mayor

Percy's Library ha sido actualizada con características avanzadas que la convierten en una aplicación de lectura de cómics verdaderamente completa y profesional.

## 📝 Sistema de Anotaciones

### Características
- **Múltiples tipos de anotaciones**:
  - 📝 Notas de texto sobre las páginas
  - 🖍️ Resaltados de áreas importantes
  - ➡️ Flechas para señalar detalles
  - ✏️ Dibujo libre
  - ⬜ Rectángulos
  - ⭕ Círculos

### Funcionalidades
- Colores personalizables para cada anotación
- Grosor de línea ajustable
- Opacidad configurable
- Sistema de tags para organización
- Búsqueda de anotaciones por texto o tags
- Exportar/importar anotaciones
- Sincronización entre dispositivos

### Uso
1. Abre un cómic y navega a la página deseada
2. Haz clic en "Anotaciones" en la barra de herramientas
3. Selecciona la herramienta que desees usar
4. Dibuja o escribe sobre la página
5. Tus anotaciones se guardan automáticamente

## 📚 Sistema de Colecciones

### Características
- Crea colecciones personalizadas de cómics
- Organiza por series, géneros, autores, etc.
- Portadas personalizables para cada colección
- Colores distintivos para identificación rápida
- Tags y descripciones detalladas
- Reordenamiento mediante drag & drop

### Funcionalidades
- Agregar cómics a múltiples colecciones
- Búsqueda dentro de colecciones
- Estadísticas por colección (total de cómics, páginas leídas, etc.)
- Exportar colecciones como listas
- Importar colecciones desde archivos
- Vista en cuadrícula o lista

### Uso
1. Ve a "Colecciones" en el menú principal
2. Crea una nueva colección con el botón "+"
3. Dale un nombre y descripción
4. Agrega cómics arrastrándolos o usando el botón "Agregar"
5. Organiza tus colecciones según tus preferencias

## 📤 Exportador de Páginas

### Formatos Soportados
- **PNG**: Mejor calidad, ideal para impresión
- **JPEG**: Buena calidad con menor tamaño
- **WebP**: Excelente compresión moderna
- **PDF**: Documento multipágina (próximamente)

### Opciones de Exportación
- **Rango flexible**:
  - Todas las páginas
  - Página actual
  - Rango personalizado (ej: 1-50)
  - Páginas seleccionadas

- **Control de calidad**:
  - Ajuste de calidad JPEG/WebP (50-100%)
  - Redimensionamiento con ratio preservado
  - Límites de ancho/alto máximos

- **Organización**:
  - Patrones de nombres personalizables
  - Exportación a ZIP
  - Metadatos incluidos

### Funcionalidades Avanzadas
- Estimación de tamaño antes de exportar
- Barra de progreso en tiempo real
- Exportación por lotes
- Watermarks personalizados (próximamente)
- Optimización automática de imágenes

### Uso
1. Abre el cómic que deseas exportar
2. Ve a "Herramientas" → "Exportar Páginas"
3. Selecciona el rango y formato deseado
4. Configura las opciones de calidad
5. Elige la carpeta de destino
6. Haz clic en "Exportar"

## 🔍 Comparador de Versiones

### Características
- Compara dos versiones del mismo cómic
- Detección automática de diferencias
- Análisis página por página
- Porcentaje de similitud
- Identificación de páginas faltantes o modificadas

### Funcionalidades
- **Comparación visual**: 
  - Vista lado a lado
  - Overlay con deslizador
  - Diferencias resaltadas

- **Análisis detallado**:
  - Páginas idénticas
  - Páginas diferentes
  - Páginas faltantes en cada versión
  - Reporte de similitud general

- **Exportación de resultados**:
  - Reporte en texto
  - Reporte en PDF
  - Imágenes de diferencias

### Uso
1. Ve a "Herramientas" → "Comparar Versiones"
2. Selecciona el primer cómic
3. Selecciona el segundo cómic
4. Elige el rango de páginas a comparar (opcional)
5. Inicia la comparación
6. Revisa el reporte de diferencias

### Casos de Uso
- Comparar versiones originales vs remasterizadas
- Verificar traducciones diferentes
- Detectar censura o modificaciones
- Comparar ediciones especiales vs regulares
- Control de calidad en colecciones

## ❓ Preguntas Frecuentes

**P: ¿Las anotaciones modifican el archivo original?**
R: No, las anotaciones se guardan en una base de datos separada.

**P: ¿Puedo compartir mis colecciones?**
R: Sí, puedes exportar colecciones como archivos JSON.

**P: ¿Los logros se pueden resetear?**
R: Los logros son permanentes, pero puedes ocultar notificaciones.

**P: ¿Puedo exportar páginas de PDFs?**
R: Sí, todas las funciones funcionan con PDFs.

**P: ¿Hay límite de anotaciones por página?**
R: No hay límite técnico, pero muchas anotaciones pueden afectar el rendimiento.
